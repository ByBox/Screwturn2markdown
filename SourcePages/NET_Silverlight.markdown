##  Scenario 
* Le Rich Internet Applications (**RIAs**) sono applicazioni web che hanno alcune delle caratteristiche delle applicazioni desktop, sono tipicamente distribuite attraverso web browser plug-in, javascript frameworks o Virtual Machines. Esempi di frameworks RIA che richiedono browser extensions (plug-in o VM) includono **Adobe Flash, Java/JavaFX e Microsoft Silverlight**, mentre esempi di frameworks RIA che fanno uso di JavaScript includono GWT e Pyjamas.
* **Silverlight** è un ambiente di Runtime sviluppato da Microsoft per piattaforme Windows e Mac che consente di visualizzare all'interno del browser applicazioni multimediali ad alta interattività. Per le altre piattaforme come quelle basate sul kernel Linux è stato rilasciato da parte di Novell una implementazione opensource chiamata Moonlight.


##  Tecnologie, Strumenti, Overview 
* Nella prima versione (1.0) Silverlight espone i dati attraverso delle API simili al DOM del browser ed è dunque possibile programmarne le interazioni con JavaScript. Questa prima versione include anche un encoder di formati video e un interprete del linguaggio XAML, la proposta di Microsoft per la progettazione di interfacce. Le versione 1.0 di Silverlight è stata rilasciata il 4 settembre 2007. Nella seconda versione (**2.0**) è stato incluso anche una versione del **Common Language Runtime** e del **Dynamic Language Runtime** che permette a Silverlight di gestire anche applicazioni scritte con un linguaggio .Net (C#, VB.NET, IronPython, IronRuby). Le versione 2.0 di Silverlight è stata rilasciata il 14 ottobre 2008.
