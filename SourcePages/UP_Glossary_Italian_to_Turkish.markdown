# Rules

* Since this glossary contains text in both Italian and Turkish languages, in order to allow both sides to understand each other, it is best to use English as language in common for everything that is not content for glossary itself (just like this section you're reading at this very moment) 

* This glossary is meant to contain translations only **from Italian to Turkish**, please do not insert anything that would translate from Turkish to Italian. For that purpose, consider creating a new page in this wiki 

* Try to not use accented characters as much as possible, this will ease a lot search&amp;find in case one of the both sides doesn't have those characters on their keyboard 

* If some text goes in a logically separated section, consider creating a new section for that purpose (look [Days of the week](IframeEditor.aspx#Days_of_the_week_1) section for an example) 

* Everything else must go under alphabetical order
