All'interno di questa videata è possibile configurare le note che verrano poi inserite o nella scheda prestazionale o nella scheda prestazionale o in entrambe.<br/>

Nella videata è possibile eseguire le operazioni di inserimento, modifica e cancellazione di ogni singola riga.<br/>
I ruoli che possono vedere tale videata sono l'amministratore il validatore.<br/>
La videata non è visibile a lista ma esclusivamente a dettaglio questo per consentire all'utente una visione completa della nota.<br/>

![](../Upload/PM/vnote.png)
</td></tr></table>


Specifichiamo meglio ogni singolo campo della videata:

* **Progressivo**: campo gestito dal sistema;
* **Codice**: _campo obbligatorio_, campo dove inserire un codice veloce di riconoscimento della nota;
* **Data inizio**: _campo obbligatorio_, se lasciato vuoto il sistema inserirà in automatico la data odierna;
* **Data fine**: è possibile inserire la data di fine validazione della nota, data in cui non verrà più visualizzata come possibile inserimento nelle schede;
* **Flag scheda Tecnica**: valore SI o NO, valore SI se può essere inserita nella scheda tecnica, NO se non verrà visualizzata;
* **Flag scheda Prestazionale**: valore SI o NO, valore SI se può essere inserita nella scheda prestazionale, NO se non verrà visualizzata;
* **Descrizione**: _campo obbligatorio_, campo descrittivo completo della nota;
* **Descrizione inglese**: campo descrittivo completo della nota in lingua inglese.
