Quando si effettua il download di un file da Internet potrebbe non essere richiesto di specificare il percorso in cui salvare il file scaricato, ma potrebbe verificarsi l'apertura diretta del file. 
Il documento esportato in excel dalla toolbar del preview a volte si apre direttamente in explorer senza avere la possibilità di salvarlo su file system.

###  Cause 
Tutti i meccanismi di download sono regolati dal sistem operativo / browser del client, nel tuo caso da windows (più precisamente in esplora risorse) si configura il tutto. ;)
